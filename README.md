A simple Age Calculator web application built using HTML, CSS, and JavaScript.
It allows users to enter their date of birth and instantly calculates their current age in years.

Technologies Used....
HTML5 – Structure of the web page
CSS3 – Styling and layout
JavaScript – Age calculation logic

How It Works.....
The user selects a date of birth
JavaScript compares the birth date with the current date
The difference in years is calculated
If the birthday hasn’t occurred yet this year, age is reduced by 1
